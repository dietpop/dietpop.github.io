Hi there!

I'm a Software Developer learning Ruby on Rails.

I've spent the past 8 years working on my business, eSeller Street, Inc., which develops sites that help buyers search eBay, Amazon, and Etsy by item popularity. We earn a commission on each purchase.

I built the first site on the LAMP stack. My background is in embedded SW development using Assembly and C language code for ADSL modems and telecom equipment. I self-studied PHP, SQL, JS, HTML, CSS, REST APIs, and server administration to release an eBay research service in 2006.

We shifted our focus to building buyer sites instead of seller tools when Affiliate commissions outperformed our paid service.

My Sites

WatchedItem.com was released in 2008 as a tool to sort eBay items by the number of people watching a listing. The Watchers sort is not available on the eBay.com site.

PopWatchers.com has WatchedItem's features but also includes Amazon and Etsy items. It really took off when Google first released their image search.

WonderFinds.com has some neat features. We use Modernizr for responsive design, i.e. the items displayed fit the user's device. Also, we run a GO algorithm behind the Python script to make square item photos so the page is easier to browse.
